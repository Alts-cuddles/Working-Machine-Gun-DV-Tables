# Working Machine Gun DV Tables

A Foundry VTT module for **Cyberpunk RED** that automatically posts a helpful reminder when you roll Autofire with a Machine Gun.

## Features

- Shows current distance and the correct DV from the Machine Gun table
- Displays how much you beat (or missed) the DV by
- Calculates the correct autofire multiplier (capped at 5x)
- Clearly shows **MISSED** when your roll does not beat the DV
- Excludes Assault Rifles and SMGs
